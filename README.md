# Compte-minute

it's a countdown or timer program for linux terminal


install ncurse and pulse audio headers files

then

type make

and type ./Compte-minute [secondes]
or type ./Compte-minute [minutes] [secondes]

you can pause, and their is a sound at the end of the countdown

you can copy the Compte-minute program into your bin directory

you can pause and exit the countdown, their is a sound at the end of the countdown.
